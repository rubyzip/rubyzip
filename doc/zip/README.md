# Collected ZIP documentation

This folder contains various useful documents relating to ZIP archives and the ZIP specification. Where restrictions on distribution exist (or are assumed) we link to the documentation rather than copy it into this repository.

* [`APPNOTE.TXT`](https://pkware.cachefly.net/webdocs/casestudies/APPNOTE.TXT): The original definition of the Zip Format, by Phil Katz of PKWARE.
* [`appnote_iz.txt`](https://libzip.org/specifications/appnote_iz.txt): Info-ZIP's unofficial and "corrected" version of the Zip Format.
* `extrafld.txt`: Some of the known extra field types supported by various ZIP tools.
* [`winzip_aes.md`](https://www.winzip.com/en/support/aes-encryption/): Details of the AES encryption method for ZIP archives.
* `ziplimit.txt`: Details of the various limits found within the ZIP specification, such as maximum comment length, etc.

Most of the above can also be found on the [libzip specifications page](https://libzip.org/specifications/).
