# Collected ZIP documentation

This folder contains various useful documents relating to ZIP archives and the ZIP specification.

* `appnote_iz.txt`: Info-ZIP's unofficial and "corrected" version of PKWARE's [`APPNOTE.TXT`](https://pkware.cachefly.net/webdocs/casestudies/APPNOTE.TXT) file.
* `extrafld.txt`: Some of the known extra field types supported by various ZIP tools.
* `winzip_aes.md`: Details of the AES encryption method for ZIP archives. Also available [here](https://www.winzip.com/en/support/aes-encryption/).
* `ziplimit.txt`: Details of the various limits found within the ZIP specification, such as maximum comment length, etc.

*As far as I know the above files do not have any restrictions on redistribution, so I copy them here for ease of finding them when needed.*
